# @jb0xyz/create-vivere

The interactive project scaffolder for [Vivere](https://github.com/jb0xyz/vivere), a TypeScript framework for Discord bots.

## Usage

```sh
npm create @jb0xyz/vivere@latest
```

It asks for a language (English, 한국어, 日本語), a transport (Gateway or HTTP), and which examples to include, then generates a ready-to-run bot project. Each project ships with an `AGENTS.md` that documents its conventions for coding agents.

Full documentation lives in the [Vivere repository](https://github.com/jb0xyz/vivere).
